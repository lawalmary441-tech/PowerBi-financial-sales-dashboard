# Financial Sales Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive sales dashboard developed using Microsoft Power BI. The dashboard provides insights into sales performance, profit, gross sales, units sold, and business trends. It is designed to help business stakeholders monitor key performance indicators (KPIs) and make data-driven decisions.

## 🎯 Objectives

- Build an interactive Power BI dashboard.
- Visualize key business performance metrics.
- Analyze sales trends over time.
- Compare sales and profit across products and countries.
- Enable interactive filtering using slicers.

## 📊 Dashboard Features

- KPI Cards
  - Total Sales
  - Total Profit
  - Gross Sales
  - Units Sold

- Visualizations
  - Sales Trend by Month
  - Sales by Product
  - Profit by Country
  - Sales by Segment

- Interactive Features
  - Year Slicer
  - Country Slicer
  - Product Slicer
  - Page Navigation between Overview and Sales Details
